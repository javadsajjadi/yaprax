پێشەکی

CSS بەکاردێت بۆ کۆنترۆڵکردنی شێوازی بەڵگەنامەی وێب بە شێوەیەکی سادە و ئاسان.

CSS کورتکراوەی “Cascading Style Sheet”ە. ئەم پەرتوکە هەردوو وەشانی CSS1,CSS2 و CSS3 دەگرێتەوە، و تێگەیشتنێکی تەواو لە CSS دەدات، لە بنەماکانییەوە دەستپێدەکات تا چەمکە پێشکەوتووەکان.

بۆچی فێری CSS بین؟

Cascading Style Sheets کە بە کورتە بە CSS ناودەبرێت، زمانێکی دیزاینی سادەیە کە مەبەستی ئاسانکردنی پرۆسە کردنی لاپەڕەکانی وێب بە شێوەیەکی پێشکەشکارانەیە.

CSS پێویستە بۆ خوێندکاران و پیشەییەکانی کارکەر بۆ ئەوەی ببنە ئەندازیارێکی نەرمەکاڵا، بە تایبەتی کاتێک لە باوشی گەشەپێدانی وێبدا کاردەکەن. هەندێک لە سوودە سەرەکییەکانی فێربوونی CSS بەمجۆرەیە:

دروستکردنی ماڵپەڕەیێکی سەرسوڕهێنەر – CSS مامەڵە لەگەڵ دیمەن و هەستکردن بە بەشێک لە لاپەڕەیەکی وێب دەکات. بە بەکارهێنانی CSS دەتوانیت ڕەنگی دەقەکە، شێوازی فۆنتەکان، دووری نێوان پەرەگرافەکان، چۆنیەتی قەبارە و داڕشتنی ستوونەکان، چ وێنەیەکی پاشبنەما یان ڕەنگێک بەکاردەهێنرێت، دیزاینی نەخشە،گۆڕانکارییەکانی پیشاندانی بۆ ئامێرە جیاوازەکان و قەبارەی ڕوومێزی چاودێری بکەیت، هەروەها کاریگەری جۆراوجۆری تر.

ببە بە وێب دیزاینەر – ئەگەر دەتەوێت وەک وێب دیزاینەرێکی پیشەیی دەست بە ئیش بکەی، دیزاینکردنی HTML و CSS کارامەیییەکی پێویستە.

چاودێری وێب – CSS فێربوون و تێگەیشتنی ئاسانە بەڵام چاودێری بەهێز بەسەر پێشکەشکردنی بەڵگەنامەیەکی HTML دابین دەکات. زۆرترین جار CSS لەگەڵ زمانەکانی مارکاپ 1HTML یان XHTML تێکەڵ دەکرێت.

فێری زمانەکانی تر ببە – کاتێک لە بنەماکانی HTML و CSS تێدەگەیت ئەوا تەکنەلۆژیا پەیوەندیدارەکانی تری وەک javascript، php، یان angular ئاسانتر دەبن بۆ تێگەیشتن.
